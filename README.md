## 📚 Repository Structure

The Lumina AI Framework is organized into multiple focused repositories:

### Core Framework
- **[lumina-core](https://github.com/mlesnews/lumina-core)** - Foundation framework, AIOS kernel, and core libraries

### Scripts & Automation
- **[lumina-scripts-ai](https://github.com/mlesnews/lumina-scripts-ai)** - AI agents, ML scripts, and LLM integration
- **[lumina-scripts-automation](https://github.com/mlesnews/lumina-scripts-automation)** - System automation and workflow orchestration
- **[lumina-scripts-integration](https://github.com/mlesnews/lumina-scripts-integration)** - API integrations and external services

### Infrastructure & Deployment
- **[lumina-infrastructure](https://github.com/mlesnews/lumina-infrastructure)** - Docker, Kubernetes, and deployment configs
- **[lumina-cloud](https://github.com/mlesnews/lumina-cloud)** - Azure Functions and serverless deployments

### Applications & Configuration
- **[lumina-applications](https://github.com/mlesnews/lumina-applications)** - Desktop and mobile applications
- **[lumina-config](https://github.com/mlesnews/lumina-config)** - Agent configurations and service settings

### Documentation Hub
- **[lumina-index](https://github.com/mlesnews/lumina-index)** - This repository - Master documentation hub

## 🚀 Quick Start

1. **Clone the core framework:**
   \`\`\`bash
   git clone https://github.com/mlesnews/lumina-core.git
   cd lumina-core
   \`\`\`

2. **Install dependencies:**
   \`\`\`bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   \`\`\`

3. **Explore the documentation** in each repository's README
